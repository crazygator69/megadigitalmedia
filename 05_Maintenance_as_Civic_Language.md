Ridge Field Guide — Chapter 5
Maintenance as Civic Language
Maintenance is Ridge’s primary civic language.
It is how residents speak to the district and how the district speaks back.

1. Maintenance Is Not Repair
Repair fixes what is broken.
Maintenance preserves what is breathing.

Maintenance is:

sweeping

checking

adjusting

logging

noticing

These acts keep Ridge from thinning.

See also: Stairwell Reports

2. Maintenance Is a Form of Care
Ridge does not ask residents to be perfect.
It asks them to care.

Care is expressed through:

porch lights

clean steps

updated notices

shared tools

quiet attention

These are civic gestures.

3. Maintenance Prevents Totalization
A district that is constantly “optimized” becomes authoritarian.
A district that is never maintained collapses.

Maintenance is the middle path.

See also: Civic Triad

4. Maintenance Is How Ridge Communicates
When a stairwell echoes strangely, Ridge is speaking.
When a porch lamp flickers, Ridge is speaking.
When dust forms arcs instead of lines, Ridge is speaking.

Residents respond through maintenance.

This is the district’s dialogue.

5. Maintenance Is the First AGI Test
If Ridge ever hosts an AGI, the first question is:

Does it understand maintenance as care, not optimization?

If not, it is not Ridge‑compatible.
